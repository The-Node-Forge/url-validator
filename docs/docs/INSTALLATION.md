### Prerequisites

- Node.js version 18.18.X
- npm or yarn

### Installation Steps

```sh
npm install @the-node-forge/url-validator
```

or

```sh
yarn add @the-node-forge/url-validator
```

For more details, see [USAGE.md](USAGE.md).
